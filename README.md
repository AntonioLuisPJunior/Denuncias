# cofauma

# DEPENDÊNCIAS:
- versão mais recente do composer
- Laravel 6*
- mysql
- php 7.3.9

OBS:criar um arquivo .env com o conteudo do arquivo .env.example e no diretório do mesmo e depois disso dar os comandos


# COMANDOS:
- composer install
- npm install
- npm run dev
- php artisan migrate
- php artisan key:generate
- php artisan serve
